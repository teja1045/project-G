# Garbage Pickup App MVP Blueprint

## MVP Goal
- Validate booking demand, payment willingness (₹49–₹99), and operational flow.

## User App Core Features
1. Login/Signup via mobile OTP
2. Add/manage multiple addresses
3. Book pickup by waste type + slot
4. Price display (pickup charge + scrap estimate)
5. Payment: UPI or cash on pickup
6. Status tracking: Requested → Assigned → Picked up → Completed

## Delivery Partner App
- Login
- View assigned pickups
- Call customer
- Mark completed
- Upload photo proof

## Admin Panel
- View bookings
- Assign worker
- Change pricing
- View payments
- Download daily report (Excel)
- Manage users

## Tech Stack
- Flutter (Android first)
- Firebase (planned)
- Razorpay (planned)
- Node/Express APIs for MVP stubs
